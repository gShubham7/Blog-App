## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`PORT` 

`DB_URL` 

`JWT_SECRET` 

`SALT_ROUNDS` 

`GITHUB_CLIENT_ID` 

`GITHUB_CLIENT_SECRET` 

`GOOGLE_CLIENT_ID` 

`GOOGLE_CLIENT_SECRET` 

`FACEBOOK_APP_ID` 

`FACEBOOK_APP_SECRET` 

`TWITTER_CONSUMER_KEY` 

`TWITTER_CONSUMER_SECRET` 

### Clodinary env

`CLOUD_NAME` 

`API_KEY` 

`API_SECRET`
